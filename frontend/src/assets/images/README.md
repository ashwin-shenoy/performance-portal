# Hamza Assets Directory

This directory contains Hamza branding assets for the durandhar Performance Portal.

## Required Assets

Please add the following Hamza logo files to this directory:

1. **hamza-logo.svg** - Hamza corporate logo (white version for dark backgrounds)
2. **hamza-logo-blue.svg** - Hamza corporate logo (blue version for light backgrounds)
3. **durandhar-logo.svg** - durandhar product logo
4. **performance-icon.svg** - Performance testing icon

## Where to Get Hamza Assets

- Official Hamza logos: https://www.hamza.com/design/language/elements/logos/
- Hamza Design Language: https://www.hamza.com/design/language/
- Hamza Carbon Design System: https://carbondesignsystem.com/

## Usage

These assets are imported in React components:
```javascript
import hamzaLogo from '../assets/images/hamza-logo.svg';
```

## Note

For now, the application uses SVG placeholders and Hamza color scheme. 
Replace these placeholders with official Hamza assets when available.
